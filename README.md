# YouTube Downloader Pro

## Requisitos Previos
- Python 3.9+
- Node.js 18+
- **FFmpeg instalado en el sistema** (necesario para conversiones de audio)

## Instalación
1. **Backend**: 
   - `cd backend`
   - `pip install -r requirements.txt`
   - `python main.py`
2. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
